# Agent Start Free Course

## Purpose
Train agents to guide prospects through a simple start free pilot that converts into a paid plan.

## Format
- 7 short modules
- Each module includes a goal, key points, and a practice task

## Module 1: Product in one minute
Goal: Explain BuildMyBot in simple language.
- BuildMyBot captures leads and answers questions 24/7.
- It is a no-code platform with chatbots and optional voice agents.
- The business keeps control of tone and brand.
Practice: Write a 20 second pitch for a local business.

## Module 2: Pick the right pilot goal
Goal: Select one clear outcome for the start free pilot.
- Common goals: more leads, faster response, reduce missed calls.
- Keep scope narrow: one website, one bot, one CTA.
Practice: Write a pilot goal for a home services company.

## Module 3: Discovery in 5 questions
Goal: Find pain and confirm fit fast.
Questions:
- How do leads reach you today?
- What happens after hours?
- What is your average response time?
- Which questions do you answer most?
- What would a good month look like?
Practice: Role-play a 5 minute discovery call.

## Module 4: The start free offer
Goal: Remove risk and get agreement.
- Frame it as a short pilot with a clear success review.
- Confirm success metrics and timeline.
Script:
"Lets start free with one bot focused on [goal]. We will review results in two weeks."
Practice: Deliver the script twice and adjust tone.

## Module 5: Demo with intent
Goal: Show only what supports the pilot goal.
- Show a fast answer to a common question.
- Capture a lead and show where it lands.
- Highlight the embed and setup speed.
Practice: Record a 5 minute demo using a test site.

## Module 6: Handle the top objections
Goal: Keep momentum without pressure.
- "We already have chat" -> "Great, this covers after hours and lead capture."
- "We need IT" -> "It is a simple script and we can share security details."
- "AI sounds risky" -> "We control the tone and can review responses."
Practice: Create a response for each objection in your own words.

## Module 7: Close and follow up
Goal: Set the next step before ending the call.
- Confirm timeline and owner.
- Send a short recap email within 2 hours.
Practice: Write a recap email in 5 sentences.

## Start free launch checklist
- Confirm goal and CTA
- Gather FAQ and top services
- Set brand colors and tone
- Embed the script on the website
- Test lead capture end to end
- Review in 7 to 14 days
