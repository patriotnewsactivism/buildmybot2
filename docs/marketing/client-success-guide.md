# Client Success Guide

## Success goals
- Increase lead capture from existing traffic
- Reduce response time for common questions
- Improve conversion to calls or appointments

## 30-60-90 day plan
Day 1 to 30
- Launch one bot for a single goal
- Track lead volume and response accuracy
- Review chat logs weekly

Day 31 to 60
- Improve responses based on real chats
- Add new FAQ content
- Add a second bot use case if needed

Day 61 to 90
- Expand to another page or channel
- Consider voice agent for phone calls
- Review ROI and plan next quarter

## Key metrics to track
- Leads captured per week
- Response time
- Conversation to lead conversion rate
- Top questions by volume

## Optimization levers
- Improve opening prompt and CTA
- Add missing FAQs
- Tighten lead capture timing
- Improve handoff rules

## Success checklist
- Monthly performance review
- Quarterly content refresh
- One new experiment per month
