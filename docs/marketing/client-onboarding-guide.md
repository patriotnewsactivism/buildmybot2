# Client Onboarding Guide

## Goal
Launch a live bot that captures leads and answers common questions within 7 to 14 days.

## What we need from you
- Website URL and brand colors
- Top 10 FAQs
- Services and pricing overview
- Preferred call to action
- Lead capture fields (name, email, phone)

## 7 day quickstart plan
Day 1: Strategy
- Confirm primary goal
- Select one use case (sales, support, or booking)

Day 2: Content
- Provide FAQs and service list
- Upload any docs or website pages

Day 3: Bot build
- Create the bot
- Set tone and brand colors

Day 4: Lead capture
- Configure lead form fields
- Set notification rules

Day 5: Embed
- Install the script on your website
- Test on desktop and mobile

Day 6: Review
- Run test conversations
- Adjust responses and handoff rules

Day 7: Launch
- Go live
- Track the first week of leads

## Launch checklist
- Bot live on main website
- Lead capture tested end to end
- Notifications tested
- FAQ accuracy reviewed
- CTA confirmed

## Helpful tips
- Keep the first bot focused on one outcome
- Start with your most common questions
- Review chats weekly and improve answers
