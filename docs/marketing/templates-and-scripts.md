# Templates and Scripts

## Discovery call script (10 minutes)
1) Warm intro
"Thanks for the time. I want to learn how leads reach you and where response time slows things down."

2) Questions
- How do leads reach you today?
- What happens after hours?
- What is your average response time?
- Which questions come up most?
- What would a great month look like?

3) Summary
"So the main gap is [pain]. If we can solve that quickly, would you be open to a short pilot?"

## Demo script (12 to 15 minutes)
1) Show the bot on the website and ask a common question
2) Show lead capture and notifications
3) Show analytics or chat logs
4) Show setup flow and embed script
5) Close with a pilot offer

Close:
"Want me to set up a start free pilot this week?"
